
Portfólio de Jose Rafael Ramos desenvolvido como atividade prática da disciplina Fundamentos da Programação Web do curso de Análise e Desenvolvimento de Sistemas 


  Seções

- *Início* — apresentação com logo e nome
- *Sobre* — informações pessoais e acadêmicas
- *Formação* — trajetória educacional em formato de timeline
- *Portifólio* — link para repositórios no GitHub
- *Fale Comigo* — formulário de contato com validação em JavaScript

Tecnologias utilizadas

- HTML5
- CSS3 
- JavaScript puro (sem bibliotecas)
- 
Funcionalidades

- Navegação fixa com destaque automático da seção ativa
- Animação de entrada das seções ao rolar a página
- Alternância de tema claro/escuro
- Validação de formulário com mensagem de erro
- Simulação de envio com mensagem de sucesso
